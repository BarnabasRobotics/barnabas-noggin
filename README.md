# barnabas-noggin
Controller board for Barnabas-Bot.  Arduino-Uno compatible board.  

Versions:
 
 Using Nano plug-in board
 - V3: Rev 2C 
 - V3.1: Rev 2D 
 
 No nano plug-in board
 - V4: Rev 3A, 3B 
 

Changes in V3.1:
 - Remove 5V regulator
 - Add fuse protection on 5V and 3.3V
 - Add DC Driver Controller
 - Fix silkscreen text
